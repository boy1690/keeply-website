為什麼要換

  • → 企業級定價。 P4 One 起價約每人每月 $19 美金，把 Snowtrack 過去服務的獨立藝術家和小型工作室擠出市場。
  • → 強制 Perforce 帳號 + SSO。 新授權需要註冊 Perforce 認證系統——對只想管 Figma 或 Blender 檔案的創作者很不友善。
  • → 依賴 Helix Server。 P4 One 工作流必須建立在中央 Helix Server 上——這種企業基礎設施對自由工作者或 3 人工作室是殺雞用牛刀。
  • → 檔案被鎖在伺服器裡。 P4 One 把版本存在 Helix 裡，你不能直接打開 NAS 用檔案總管看你的專案。Keeply 讓所有檔案維持成普通檔案，用 Finder 或檔案總管就看得到。
  • → 失去「輕量創作」的精神。 P4 One 的路線圖明顯轉向大型工作室，離 Snowtrack 過去服務的獨立創作者族群越來越遠。

怎麼換到 Keeply

  1. 1

    趁還能用的時候匯出 Snowtrack 歷史

    如果你的 Snowtrack 還能開，打開你的專案資料夾，把 .snowtrack/ 整個目錄複製到外接硬碟。把舊版本歷史當唯讀封存保留。

  2. 2

    安裝 Keeply 並指向你的專案資料夾

    下載 Keeply、打開你的專案資料夾、點「開始保護」。Keeply 從目前狀態開始記錄版本——新的版本歷史、同樣的檔案、不需要任何轉換工具。

  3. 3

    選一個備份位置（NAS / 隨身碟 / Gitea / GitHub）

    在 Keeply → 備份位置新增一個：本機資料夾、隨身碟、NAS（SMB 路徑）、自架 Gitea、或 GitHub repo。每次存檔 Keeply 都會自動同步到那邊——不需要中央伺服器。

常見問題

Snowtrack 發生什麼事？ ▼
Perforce 在 2025 年 3 月收購 Snowtrack 並改名為 P4 One。獨立版 Snowtrack 不再販售——新授權必須訂閱 Perforce Helix Server，企業方案起價約每人每月 $19 美金。獨立創作者被迫尋找替代品。
Keeply 真的免費嗎？ ▼
是的，出貨版對個人永久免費。核心功能包括無限版本歷史、視覺比對、檔案差異、版本筆記、單一位置備份都免費。Team 方案（含管理控制與多位置同步）正在積極開發中。$599 永久授權 Founding Member（限 500 套）一次購買即可永久使用未來所有 Team 功能。
使用 Keeply 需要懂 Git 嗎？ ▼
不需要。Keeply 專為不想碰 Git 的創作者設計——「介面零 Git 術語」是我們堅守的產品原則。操作名稱用辦公室語言：儲存版本、瀏覽歷史、回復到此版本、比較版本。沒有 commit、branch、merge 要學。
Keeply 能處理大檔案嗎？像 3D、影片、Blender 專案？ ▼
可以，而且完全不用設定。Keeply 自動偵測 38 種二進位格式（Blender、PSD、影片、3D、壓縮檔等）加上任何超過 10 MB 的檔案，把它們靜默轉到大檔儲存通道。repo 不會膨脹；也不會看到別的工具讓你搞 LFS 設定那一套。
哪些檔案類型能看到並排視覺比對？ ▼
Keeply 對 Microsoft Office 文件（Word .docx、Excel .xlsx、PowerPoint .pptx）、PDF、點陣圖（PNG / JPG / WebP）都能產生真正的視覺比對。你會看到兩個版本並排、差異高亮——不是只看一串改過的 byte。其他檔案類型仍可追蹤版本，隨時回復任一時間點。
我的資料實際存放在哪裡？ ▼
你的專案檔案留在你的電腦上，備份副本可選 5 種自我託管位置：本機資料夾、隨身碟、NAS（SMB 路徑）、自架 Gitea、或 GitHub repo。Keeply 自家伺服器完全不接觸你的專案資料——只在授權驗證時收到雜湊過的裝置識別碼。
