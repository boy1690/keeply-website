1. Software

Keeply je desktopová aplikace poskytující vizuální správu verzí projektových souborů. Běží lokálně na vašem počítači a ukládá všechna data na místní zařízení nebo NAS. Keeply je postaven na technologii Git, ale je navržen pro každého, kdo v práci upravuje dokumenty — uživatelé nikdy nemusí přímo pracovat s Gitem.

3.5 Definice

"Týmový trezor" znamená sdílené úložiště — včetně, ale bez omezení na, zařízení Network Attached Storage (NAS), souborový server, vlastní Git server nebo cloudový Git remote repozitář — určené jako centrální bod spolupráce. Týmový trezor je identifikován jedinečným tokenem umístění generovaným Softwarem a uloženým v repozitáři.

"Umístění trezoru" znamená jedinečný identifikátor přiřazený Týmovému trezoru, odvozený z tokenu umístění vloženého v repozitáři a/nebo normalizované Git remote URL. Umístění trezoru se dělí na dva typy:

  • (a) "Pevné umístění" znamená Umístění trezoru nastavené administrátorem týmu a sdílené všemi členy týmu, maximálně 2;
  • (b) "Flexibilní umístění" znamená Umístění trezoru vyžádané členem týmu a schválené administrátorem pro osobní použití daného člena, maximálně 1 na člena.

3.6 Pravidla umístění trezoru

Změny Pevných umístění podléhají následujícím podmínkám:

  • (a) Čekací období sedmi (7) dnů od poslední změny;
  • (b) Maximálně 6 změn za rok;
  • (c) Migrace dat repozitáře (včetně vloženého tokenu umístění) na nový úložný hardware nepředstavuje změnu Pevného umístění, pokud token umístění zůstane nezměněn.

Flexibilní umístění jsou vyžádána členy prostřednictvím Softwaru a nabývají účinnosti po schválení administrátorem. Administrátor může kdykoli zrušit Flexibilní umístění jakéhokoli člena. Flexibilní umístění, ke kterým nebylo přistoupeno 30 po sobě jdoucích dnů, jsou automaticky uvolněna.
