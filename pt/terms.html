3.5 Definições

"Cofre de Equipa" designa um local de armazenamento partilhado — incluindo, mas não se limitando a, um dispositivo de Network Attached Storage (NAS), servidor de ficheiros, servidor Git auto-hospedado ou repositório Git remoto na nuvem — designado como ponto central de colaboração. Um Cofre de Equipa é identificado por um token de localização único gerado pelo Software e armazenado no repositório.

"Localização de Cofre" designa o identificador único associado a um Cofre de Equipa, derivado do token de localização incorporado no repositório e/ou do URL Git remoto normalizado. As Localizações de Cofre classificam-se em dois tipos:

  • (a) "Localização Fixa" designa uma Localização de Cofre definida pelo administrador da equipa e partilhada por todos os membros da equipa, até um máximo de 2;
  • (b) "Localização Flexível" designa uma Localização de Cofre solicitada por um membro da equipa e aprovada pelo administrador para uso pessoal desse membro, até 1 por membro.

3.6 Regras de Localização de Cofre

As alterações às Localizações Fixas estão sujeitas ao seguinte:

  • (a) Um período de espera de sete (7) dias desde a alteração mais recente;
  • (b) Não mais de 6 alterações por ano;
  • (c) A migração de dados do repositório (incluindo o token de localização incorporado) para novo hardware de armazenamento não constitui uma alteração de Localização Fixa, desde que o token de localização permaneça inalterado.

As Localizações Flexíveis são solicitadas pelos membros através do Software e entram em vigor após aprovação do administrador. O administrador pode revogar a Localização Flexível de qualquer membro a qualquer momento. As Localizações Flexíveis não acedidas durante 30 dias consecutivos são automaticamente libertadas.
